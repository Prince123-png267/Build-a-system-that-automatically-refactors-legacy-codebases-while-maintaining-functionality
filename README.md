# 🤖 AI Refactor Agents

An intelligent multi-agent system that automatically analyzes legacy code, detects issues, improves code quality, and ensures functionality through automated testing.

---

## 🚀 Overview

This project is designed to simplify code maintenance by using multiple AI agents that work together to:

* Analyze code quality
* Detect security vulnerabilities
* Refactor outdated patterns
* Provide intelligent suggestions
* Test functionality after refactoring

---

## 🧠 Architecture

The system follows a **multi-agent architecture**, where each agent performs a specific task:

* 🔍 **Analyzer Agent** → Detects bad coding practices
* 🔐 **Security Agent** → Identifies vulnerabilities
* ✨ **Refactor Agent** → Automatically fixes code
* 🤖 **Suggestion Agent** → Provides improvements
* 🧪 **Test Agent** → Validates code execution

---

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/your-username/ai-refactor-agents.git
cd ai-refactor-agents
```

2. Install dependencies:

```
pip install streamlit
```

3. Run the application:

```
python -m streamlit run app.py
```

## 🧪 How It Works

1. Upload a Python file
2. Click **Run AI Agents**
3. The system will:

   * Detect issues
   * Identify security risks
   * Refactor the code
   * Provide suggestions
   * Test the updated code
